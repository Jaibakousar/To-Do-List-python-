# To-Do List

## Description
A simple command-line To-Do List application built with Python. It allows users to add, view, and remove tasks through a menu-driven interface.

## Features
- Add tasks
- View tasks
- Remove tasks
- Menu-driven interface

## Technologies Used
- Python 3

## How to Run
1. Clone this repository.
2. Open a terminal in the project folder.
3. Run:
   python main.py

## Sample Output
===== TO-DO LIST =====
1. Add Task
2. View Tasks
3. Remove Task
4. Exit

- Save tasks to a file
- Mark tasks as completed
